本项研究构建了一套高仿真、高精度的京沪高铁全线动态调度运行图可视化系统。该系统深度集成京沪高铁全量时刻表数据，利用Python编程语言精准模拟列车在1318公里物理空间内的实时位移状态。其核心突破在于实现了极致的视觉颗粒度，通过毫秒级线性插值算法打破了传统的分钟级跳变限制，使列车点位在车站间呈现出匀速、丝滑的滑动轨迹。在视觉逻辑上，系统根据运行方向与状态自动切换形态。蓝色与红色圆点分别对应上、下行车次，绿色圆点则依据时刻表实时展示列车的到发停靠过程。针对北京南、上海虹桥等高密度枢纽站，系统自研了“终到整备”排队模块，使紫色方块模拟列车进站后的整备清场，并实现多车次横向自动排队，有效解决了高频调度下的视觉堆叠问题。通过人工拉伸上海枢纽空间坐标以及站名动态避让技术。
